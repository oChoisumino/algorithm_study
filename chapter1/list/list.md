# Linked List
### * 기본 개념
* 노드 : 리스트 내의 각 요소
* linked list : 노드를 연결해서 만드는 리스트
* 데이터 필드 + 연결 포인터 필드 로 구성

### * 주요 연산
* 노드 생성/소멸
* 노드 추가
* 노드 탐색
* 노드 삭제
* 노드 삽입

# 더블 링크드 리스트
### * 기본 개념
* 양방향 탐색이 가능한 linked list
* prev 연결 포인터 필드 + 데이터 필드 + next 연결 포인터 필드

### * 기본 개념
