도도는 이번 신촌캠프에서 시간복잡도의 개념에 대해 배웠다. 
하지만 듣다가 졸려서 자버린 결과 오개념을 가져 버렸는데, 바로 반복문의 개수로만 시간복잡도를 판단한다는 것이다. 

시간복잡도를 판단하기 위해 도도는 같은 줄에 몇 개의 반복문이 있는지 궁금해졌다. 
도도가 반복문으로 생각하는 코드는 for, while 이다. 

도도를 위해 같은 줄에 있는 반복문의 최대 개수를 구하는 프로그램을 작성해보자.

첫째 줄에 코드의 줄 개수 C (1 <= C <= 100)가 주어진다.

둘째 줄부터 C개의 줄에 걸쳐 코드가 입력된다. 입력으로 주어지는 코드는 모두 알파벳 소문자이다. 
각 줄은 10,000자를 넘지 않는다.

첫째 줄에 같은 줄에 있는 반복문의 최대 개수를 출력한다.

    입력                                        출력
     3                                           4
    for
forwhileforfor
